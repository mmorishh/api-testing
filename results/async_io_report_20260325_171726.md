# Async I/O Performance Report

**Date:** 2026-03-25T17:17:26.288825

## Results (50 requests with 100ms delay)

| Framework | Mode | RPS | Duration (s) |
|-----------|------|-----|--------------|
| FASTAPI | sync | 7.6 | 6.56 |
| FASTAPI | async | 223.4 | 0.22 |
| FLASK | sync | 7.6 | 6.58 |
| FLASK | async | 28.4 | 1.76 |

## Key Finding

**FastAPI handles async I/O 7.9x better than Flask.**

- FastAPI async: 223.4 RPS
- Flask async: 28.4 RPS

### Interpretation

- **FastAPI (ASGI)**: Async model allows handling many concurrent I/O operations without blocking threads
- **Flask (WSGI)**: Sync model blocks while waiting for I/O, limiting concurrency
- **Recommendation**: For I/O-heavy APIs (database calls, external services), FastAPI provides significantly better performance
